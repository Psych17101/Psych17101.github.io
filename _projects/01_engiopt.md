As an MSc semester project with the **IDEAL Lab** (AI in Engineering Design), I extended generative models — GANs, SliceGAN, and a multi-variable Variational Autoencoder — from 2D to the **3D domain** for engineering-design generation.

The work was contributed upstream to **[EngiOpt](https://github.com/IDEALLab/EngiOpt)**, an open-source engineering-design optimisation library, through a maintainer-reviewed pull request ([IDEALLab/EngiOpt #43](https://github.com/IDEALLab/EngiOpt/pull/43), ~2,100 lines). I iterated the code to pass the project's `ruff`, `mypy`, and pre-commit checks and addressed reviewer feedback.
